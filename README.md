# Commit Protocol

Immutable on-chain commits. Irreversible by design.

Commit Protocol is a minimal on-chain protocol that allows users to permanently
record human-readable messages directly on the blockchain.

Once committed, data cannot be edited, deleted, or overwritten.

## Key Properties

- Fully on-chain storage
- Immutable by design
- No admin keys
- No upgradeability
- No moderation
- No backend
- No custody of funds

---

## Network

- **Chain:** Base Mainnet
- **Standard:** ERC-721
- **Version:** v0
- **Commit Fee:** Fixed (see contract)

---

## Usage

Users interact with the protocol through a static front-end or directly
via the smart contract.
